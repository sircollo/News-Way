# News-Way
## Author

[Collins Sirwani](https://github.com/sircollo)

## Description

This is a python Flask web application that uses NewsAPI to help a user list and preview news articles from various sources.


## Requirements
* python3.8
* pip
* Flask
* Bootstrap

## Installation / Setup instruction

* Open Terminal 

* git clone ```https://github.com/sircollo/News-Way.git```

* cd News-Way

* code .

### Running the Application
* To run the application, open the cloned file in terminal and run the following command:

        $ ./start.sh
* To run tests:

        $ python3.8 manage.py tests
